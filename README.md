Campus Connect is a student-focused web application that provides event management, announcements, and campus resource access in one centralized platform to improve communication and engagement.

Problem statement:
Many students struggle to stay informed about campus events, academic announcements, and available support services due to scattered communication channels. Important information is often shared across multiple platforms, leading to confusion, missed opportunities, and low student engagement. There is a need for a centralized digital system to streamline communication and improve accessibility for students.
